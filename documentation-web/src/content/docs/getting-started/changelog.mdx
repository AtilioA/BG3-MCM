## Version 1.x <Badge text="Current" variant="success" />

[View Releases](https://github.com/atilioa/bg3-mcm/releases)

Key features across versions:

- **Keybinding Management** (v1.19+) - Built-in keybinding system with conflict detection
- **Event Buttons** (v1.17+) - Configurable buttons that trigger Lua callbacks
- **List V2** (v1.17+) - Modern list editor with search, reordering, and enable/disable
- **Store API** (v1.40+) - Persistent key-value storage outside the blueprint system
- **Multiple Profiles** - Create, save, load, and delete configuration profiles
- **Localization Support** - Translate settings into different languages
- **40+ Automated Tests** - Robust validation and error reporting
