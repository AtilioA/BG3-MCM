### Requirements

<Steps>
  1. **Baldur's Gate 3** (with latest patch)
  2. **BG3 Script Extender (BG3SE)** - required for MCM to function
  3. **MCM mod** installed from [Nexus Mods](https://www.nexusmods.com/baldursgate3/mods/9162)
</Steps>

### Installation for players

<Steps>
  1. Download MCM from [Nexus Mods](https://www.nexusmods.com/baldursgate3/mods/9162)
  2. Install using your preferred mod manager (Vortex, BG3 Mod Manager, etc.)
  3. Ensure MCM loads before other mods that depend on it (most mod managers handle this automatically)
</Steps>

### Installation for mod authors

To integrate MCM into your mod:

<Steps>
  1. **Add MCM as a dependency** in your mod's `meta.lsx` file. Use the current MCM UUID `755a8a72-407f-4f0d-9a33-274ac0f0b53d` and set `Version64` to the minimum version you support. See a canonical dependency example in [Mod Uninstaller meta.lsx](https://github.com/AtilioA/BG3-mod-uninstaller/blob/main/Mod%20Uninstaller/Mods/ModUninstaller/meta.lsx#L7-L24).
  2. **Create an `MCM_blueprint.json`** file alongside your `meta.lsx`
  3. **Set up IDE support** (optional but recommended): Add the [JSON schema](https://raw.githubusercontent.com/atilioa/bg3-mcm/main/.vscode/schema.json) to your VS Code settings for autocomplete and validation
  4. **Replace hardcoded values** with `MCM.Get("settingId")` calls in your Lua code
</Steps>

### IDE setup for blueprint authoring

For autocomplete and validation in VS Code:

1. Press `Ctrl+Shift+P` and select "Open User Settings (JSON)"
2. Add this to your settings:

```json
"json.schemas": [
  {
    "fileMatch": ["MCM_blueprint*.json"],
    "url": "https://raw.githubusercontent.com/atilioa/bg3-mcm/main/.vscode/schema.json"
  }
]
```
