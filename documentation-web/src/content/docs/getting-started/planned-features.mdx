## Roadmap

Here are features and improvements being considered for future MCM releases:

### In Progress

- **Improved Store API** - More storage backends and better synchronization
- **Enhanced List Features** - Better filtering and bulk operations

### Planned Enhancements

- **Gamepad Support** - Navigate MCM with a controller
- **Dynamic Blueprint Updates** - Refresh UI without game restart
- **Nested Structures** - Support for more complex tab/section hierarchies
- **Blueprint Migration Tooling** - Helpers to upgrade old blueprints

### Community Suggestions

- **Slider Labels** - Custom labels for slider values
- **Color Presets** - Save and load color configurations
- **Profile Import/Export** - Share settings between players
- **Per-Character Profiles** - Different settings per save file
